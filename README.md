### BASE Express Framework with fully necessary libraries

### Requirement

- [Node JS](https://nodejs.org)

- [Express JS](https://www.npmjs.com/package/express)

### Element

- ExpressJS

### Libraries

- Nodemon (Hot update)

- Morgan (HTTP logger)  

- Express Handlebar (template engine)

- Tailwindcss (CSS)

- Vite

